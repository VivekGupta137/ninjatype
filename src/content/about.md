# <i class="material-icons">info</i> About NinjaType

NinjaType is a minimalist typing interface designed to help you improve your typing speed and accuracy. Built with modern web technologies, it provides a clean, distraction-free environment for practicing your typing skills.

## Technology Stack

NinjaType is built using:
- **Astro** - Fast, content-focused web framework
- **React** - Interactive UI components
- **TypeScript** - Type-safe development
- **Nanostores** - Lightweight state management
- **Cloudflare Pages** - Fast, global deployment
